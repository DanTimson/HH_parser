# HH_parser

## Quick start
### Windows
```commandline
git clone https://github.com/Kitsumetri/HH_parser.git
cd HH_parser
py -m venv .venv
.venv\scripts\activate
pip3 install -r requirements.txt
```

### Linux
```commandline
git clone https://github.com/Kitsumetri/HH_parser.git
cd HH_parser
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
